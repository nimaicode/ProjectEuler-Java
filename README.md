# Project Euler – Java Problem Solving Journey

Welcome to my Project Euler repository!  
This project is a personal collection of solutions to mathematical and algorithmic problems from Project Euler, solved using **Java**.

The goal is to strengthen my problem-solving skills, improve my understanding of algorithms, and build a consistent coding portfolio over time.

In this repository, I:

- Keep each problem in a single `.java` file for simplicity
- Focus on clean, efficient, and readable code
- Continuously improve algorithmic thinking and performance
